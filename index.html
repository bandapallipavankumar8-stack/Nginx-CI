VPC (Virtual Private Cloud)Isolated logical virtual networks to launch isolated cloud infrastructure.

🐙 GitHub Core Team Actions
git clone [url]Clones down a secure central code repository to local environments.
git commit -am "message"Stages all tracked mutations and builds a structural local revision history block.
git push origin [branch]Uploads verified tracking commits directly into the central cloud remote branch.
git pullFetches clean upstream iterations and merges them locally.
🐳 Docker Engine Commands
docker build -t [name] .Compiles a target container application runtime configuration from a Dockerfile layout.
docker run -d -p 80:80 [image]Triggers an isolated container daemon running in detached back-end network mode.
docker psDisplays active, working containers running on the local architecture engine layer.
docker rmi $(docker images -q)Force clears redundant stale local container cache layouts.
